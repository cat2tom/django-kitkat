views.process_qa_form(request, first_request=False)¶

Form display and save method for Daily Photon Output Test, Photon Check Calibration and Electron Check Calibration

Constructs QATestResultForm object from POST data from request. Creates all form objects (Float Value Forms, Bool Value Forms, Float Average Value Forms, Test Equipment Forms, Temperature Pressure Forms) to assemble the data entry form for the selected test and pass these to the ‘qa_testforms.html’ template.

The request contains either:
  • POST data from from a Select Machine and Test Form (first_request == True) in which case the data entry form is assembled using blank forms
  • POST data from a completed data entry form for validation and saving. If all sub-forms validate, data is saved into the database and user is redirected to the ‘display_entry’ page for the data submitted. If any form does not validate, the user is returned to the entry form with error messages
